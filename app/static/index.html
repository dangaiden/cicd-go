Simple Pokemon API

This is a simple app in GO with some stored values (Pokemon) that can be queried.

Examples:

- Go to "https://holded.itgaiden.com/all" to review all items within the slice in JSON format. - If you want to check an specific item use the URI "https://holded.itgaiden.com/pokemon/"

ADD ITEM

- To add an item (POST) you can use curl, postman or whatever tool you want. An example: curl -X POST -d '{"Name":"Pikachu","Number":"025","Type":"Electric"}' \ -H "Content-Type: application/json" \ https://holded.itgaiden.com/pokemon

REMOVE ITEM

- Remove an item use: "/pokemon/ curl -X DELETE https://holded.itgaiden.com/pokemon/004
